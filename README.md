# 🖥️ DR_Hud – Advanced FiveM HUD

DR_Hud is a modern and fully customizable HUD system for FiveM servers (ESX / QBCore / Old ESX / New ESX).

It features a Vue.js-based UI, advanced speedometer, player status system, voice indicator, minimap customization, and in-game settings menu.

---

## ✨ Features

- 🚗 Realistic Speedometer (KM/H)
- ⛽ Fuel system support
- 🔊 Voice activity indicator (talk detection)
- ❤️ Health / Armor / Hunger / Thirst / Stamina HUD
- 📍 Dynamic street name display
- 🧍 Microphone status indicator
- 🔒 Seatbelt system with crash protection
- 🧭 Custom minimap positioning
- ⏸ Pause menu detection
- ⚙️ In-game HUD settings menu (F4)
- 💰 Cash & Bank display (ESX / QBCore support)
- 🔄 Automatic framework detection

---

## 📦 Supported Frameworks

- ESX
- New ESX
- QBCore
- Old QBCore
